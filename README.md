# 🎴 杀戮尖塔类卡牌游戏（开发中）
> 基于 Unity 开发的 Roguelike 卡牌构筑游戏，致敬《杀戮尖塔》，目前处于半成品开发阶段。

---

## 📖 项目简介
本项目是一款回合制卡牌构筑Roguelike游戏，使用 Unity 引擎与 C# 开发，目标是复刻经典卡牌游戏的核心战斗与构筑体验，并加入原创卡牌、遗物与敌人机制。

目前已完成战斗系统、卡牌系统的基础框架，后续将继续扩展卡牌池、遗物系统、敌人AI、地图流程与剧情内容。

---
